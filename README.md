# Requirements
* [Node.js][nodejs-page]
* [npm][npm-page] or [yarn][yarn-page]
* [Gulp.js][gulpjs-page]

# Project Structure
```
* build/                        build result
* src/
  * html/                       html files
  * js/                         js files
  * css                         css files
* vendors                       3rd party libraries
```

```sh
$ npm install
$ npm run build
```
